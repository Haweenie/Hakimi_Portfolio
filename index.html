About Me

I am currently a student at Nanyang Polytechnic in Game Development and Technology. I have worked on multiple projects using Unity and Unreal Engine, with experience using C# and C++. Through these projects, I have come to use the skills I have learnt in programming, as well as gameplay and level design, to create games. Below is a showcase of some of the school projects I have done so far.

2D Level Design - Mario Editor Tutorial Level

A project using Mario Editor to design our own tutorial levels to introduce the various gameplay mechanics of the game.

2D Level Design - Mario Editor Levels

Another project using Mario Editor. This project expands on what we had previously learnt and had us creating 2 unique levels, each showcasing a different powerup.

For the first level, I designed a level around the shell powerup. I made use of its ability to slide and kill enemies by incorporating slopes to allow the player to slide through the level quickly and implementing optional areas that can only be accessed using the powerup.

The next level made use of the propellor mushroom, allowing Mario to fly and land on platforms higher up that are usually our of reach. Using this mechanic, I added areas specific to the powerup where players had to use it to dodge and avoid hazards.

3D Level Design - Greybox Level Design

Using Unreal Engine, I made use of the design principles I had learnt to create a greybox level where the player was tasked to travel from the start point to the end point.

3D Level Design - FPS Level Design

This project had us designing our own FPS level, with various enemy types and mechanics to make use of.
